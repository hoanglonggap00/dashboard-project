# php-learning

This project contain most of the core functionalities of a website:
1. User:
  - login 
  - register
  - change password 
  - reset password (using phpMailer)
  - remember me 
  - logout 
2. Product:
  - insert product 
  - update product info 
  - delete product 
  - set active/deactive 
  - sort/search
